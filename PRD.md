# OpenClaw 多 Agent BI 系统 PRD

## 项目概述

基于 OpenClaw 框架的多 Agent 协作智能 BI 系统 Web 端。

## 8个核心 Agent

1. BI需求Agent - 理解自然语言需求
2. BI指标设计Agent - 设计指标计算口径
3. BI数据模型设计Agent - 设计数据模型
4. BI指标ETL Agent - 生成 ETL SQL
5. BI指标服务Agent - API 接口服务
6. BI可视化Agent - 生成图表
7. BI权限管理Agent - 权限控制
8. BI运营Agent - 监控预警

## Web端功能

- 登录页
- Agent 对话界面
- 指标管理列表
- 可视化仪表盘
- 权限配置

## 技术栈

前端：React + TypeScript + TailwindCSS
后端：Node.js + Express
图表：ECharts
